# STATION-BASED-MONTHLY-DEMAND-FORECASTING-SYSTEM-FOR-CAR-RENTALS-USING-DATA-DRIVEN-ALGORITHMS_PUBLIC

This repository contains a public overview of the master’s thesis:

“Station-Based Monthly Demand Forecasting Pipeline for Car Rentals Using Data-Driven Algorithms.”

!Source Code Availability
The complete implementation and company-specific data pipelines are not publicly available due to confidentiality restrictions related to proprietary company data.
Access to the private repository containing the full source code and implementation details may be granted upon reasonable request for academic purposes.

The study addresses mid-term, station-based demand forecasting for multi-vehicle car rental services, with the objective of supporting data-driven fleet planning decisions. Due to the complex interactions between temporal patterns, vehicle categories, and business-related factors, accurate demand forecasting remains a significant industry challenge.

The proposed system implements an end-to-end forecasting pipeline that begins with retrieving historical rental data from a company database, followed by feature selection, categorical encoding, and preprocessing steps designed to increase data homogeneity. The dataset is then segmented by vehicle class, and multiple time-series and machine learning models are trained independently for each class.

A diverse set of models—including statistical methods, regression-based algorithms, and deep learning architectures—is evaluated per vehicle category. Model performance is assessed using accuracy metrics, after which a stacking ensemble approach is applied to combine the best-performing models.

This pipeline leverages real-world company data and data-driven techniques to improve robustness, scalability, and forecasting accuracy under practical operational constraints.
